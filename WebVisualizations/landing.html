Summary:

We perform analysis on weather data to demonstrate that the average temperature tends to be higher atlatitudes near the equator. The data is acquired using OpenWeatherMap API by pulling data from 500 citiesselected at random latitudes.

The open source python package, matplotlib, is used to present the relations between latitude and weatherproperties like temperature, cloudiness, wind speed and humidity. This website showcases the results andprovides written descriptions and interpretations of the data.
